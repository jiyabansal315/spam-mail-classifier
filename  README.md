# Spam Mail Classifier

A Machine Learning project that classifies emails as Spam or Ham (Not Spam) using Natural Language Processing (NLP) techniques and supervised learning algorithms.

## Dataset

The dataset contains email messages labeled as:

* Spam
* Ham (Not Spam)

## Project Workflow

1. Load and explore the dataset
2. Clean and preprocess text data
3. Convert text into numerical features using TF-IDF Vectorization
4. Split data into training and testing sets
5. Train a Machine Learning model
6. Evaluate model performance
7. Predict whether a custom email is Spam or Ham

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Natural Language Processing (NLP)
